# 项目简介

## 核心功能
- **管理员功能**
  - 个人中心
  - 基础数据管理：课程类型、器材类型
  - 健身房管理
  - 教练管理
  - 课程管理
  - 器材管理
  - 用户管理：续费会员
- **用户功能**
  - 健身房查询
  - 教练查询
  - 课程查询
  - 器材查询
  - 个人中心

## 适用场景

- 健身房信息管理
- 教练资源调度
- 课程安排与查询
- 器材管理维护
- 会员服务与续费

## 技术特点
- 采用Spring Boot框架，提高开发效率
- 使用Vue.js前端技术，实现响应式界面设计
- 数据库采用MySQL，保证数据安全与稳定
- Maven项目构建，便于项目管理和版本控制
